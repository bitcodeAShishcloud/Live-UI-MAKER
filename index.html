Quick Guide

Shortcuts

  • Run: Ctrl/Cmd + S or Ctrl/Cmd + Enter
  • Toggle preview: Ctrl/Cmd + Shift + P
  • Format file: Ctrl/Cmd + Shift + F
  • Toggle sidebar: Ctrl/Cmd + B
  • Refresh preview: F5

Guide

  • Create/import files in the sidebar.
  • Use tabs to edit; Run or Refresh to update preview.
  • Console button shows logs/errors from preview.
  • Markdown files auto-open a live MD preview tab.
  • Export icon on tabs downloads the file.
